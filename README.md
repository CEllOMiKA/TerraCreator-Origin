# TerraCreator-Public
### Milestone #1 (inDEV)
### 介绍:

一个泰拉瑞亚tModLoader模组制作器

不同于Darthsae和FunCrafter1234的tCreator

TerraCreator使用Windows Form老牌框架(old但稳定)

基于.NET8.0(与tModLoader一致)

### 系统要求:

Windows7 SP1及以上

.NET 8.0  (下载地址:https://dotnet.microsoft.com/)
